# Controle Frotas Project
REST API for Controle Frotas Web App

## Quick start

### Clone the project
```sh
git clone git@github.com:opaulochaves/api-controle-frotas.git
```

### Install all dependencies
```sh
yarn install
```

### Run in dev mode
```sh
yarn run dev
```

### Run in production mode
```sh
yarn start
```
