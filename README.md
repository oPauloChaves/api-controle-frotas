# reskongo-api
REST API for Reskongo Project Management Web App

## Quick start

### Clone the project
```sh
git clone git@github.com:paulochavesbr/reskongo-api.git
```

### Install all dependencies
```sh
yarn install
```

### Run in dev mode
```sh
yarn run dev
```

### Run in production mode
```sh
yarn start
```
